파이썬의 가장 기초인 합산에 대해 설명을 하겠습니다 파이썬 합산을 위해선 **숫자형**을 
알아야하는데 그중에서 기본적인 숫자형을 알려드리겠습니다.

```py
print(3+7) #더하기
a=7 #변수
b=3 #변수
print(a+b) #더하기
print(a/b) #나누기
print(a//b) #나눈 횟수
print(a%b) #나눈 나머지의값

```

이것만 기본적으로 외우면 쉽게 합산 문제를 해결할수있습니다  
그리고 합산문제에 필요한 계속 더하기에 대해 알려드리겠습니다

```py
a=0 #변수
while a<7: #반복문 (a가 7보다 작으면)
    a=a+1 #a에다가 a를 더하여 값 상승
print(a) # 1씩 더한값을 출력
```

숫자형과 반복문을 완벽하게 이해를 하셨다면
***합산문제*** 를 푸실수있습니다

문제: 1부터 10까지의합산

```py
a=0 #변수
b=0 #변수
while a<10: #a가 10보다 작을때 반복(반복문
    a=a+1 #a 값 증가
    b=b+a #b라는 변수에다가 a 의값 '1'을 저장
print(b) #반복이 끝난후 최종값을 출력
```
---
이런식으로 반복 합산문제를 손쉽게 푸실수있습니다
